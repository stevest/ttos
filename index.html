    Welcome to ttos project page.

    This project is aiming to augment the already packed TreesToolbox toolbox for Matlab, making easy to handle multiple morphologies. This is done using an natural, object oriented way. Using the ttos help you:

    1. Load multiple dendritic trees; group them into cells.
    2. Easily swap cell regions (ex soma, dendrites) between cells.
    3. Prune and trim the cell with ease (comming soon).
    4. Enhanced plotting tools
    5. Requests are welcome

    Begin by downloading an SWC morphologies straight from Neuromorpho.org and loading them into new cell objects:

    >> mycell_A = ncell([],1,'myfile_A.swc');
>> mycell_B = ncell([],1,'myfile_B.swc');

    Then you can get a specific region, like apicals:

    >> apicals = mycell_A.getSubtree('apic');

    And swap them in cell B:

    >> mycell_B.swapSubtree(apicals,'apic');;

    Of course there are numerus other tricks to try, so hurry on project's Wiki.
